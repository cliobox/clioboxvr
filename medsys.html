Evolución

MedSys empezó su desarrollo en el año 2000 como un poderoso sistema de gestión médica. Desde sus inicios, la principal característica de MedSys siempre fue la facilidad de uso y las pantallas minimalistas, siendo uno de las primeras aplicaciones para PC desarrolladas con la misma filosofía de las páginas web actuales. Continuando su evolución, hemos creado MedSys con versiones para médicos generales/cirujanos, ginecólogos y odontólogos y próximamente para pediatras… ven y conoce más de MedSys.

MedSys actualmente está disponible en versiones para Médicos Generales/Cirujanos, Ginecólogos/Obstetras y Odontólogos… y próximamente para Pediatras.

MedSys para Médicos Generales y Cirujanos

Este módulo se incluye en todos los otros módulos y permite la gestión de historias clínicas, pacientes, consultas, impresión de recetas, control de citas, agenda, imágenes, ingresos, egresos y estados de cuenta, generación de certificados y documentos, impresión de sobres membretados, etc, etc. Este módulo fundamental puede adaptarse por sí mismo a cualquier especialidad médica.

MedSys para Odontólogos

El nuevo módulo para odontólogos permite el ingreso gráfico de odontogramas y es el único sistema que contiene el diseño de odontograma aprobado por el Ministerio de Salud del Ecuador, donde además el medico podrá llevar el control de enfermedades periodontales, mal oclusión, fluorosis, índices CPO-ceo e higiene oral simplificada con cálculos automáticos.

MedSys para Ginecólogos y Obstetras

El módulo para ginecólogos y obstetras permite crear tablas de fertilidad con fechas de ovulación, desfases y periodos no fértiles, así como tablas obstétricas con predicciones de fecha de parto y cada una de las fechas críticas dentro de todo el proceso de embarazo, como movimientos fetales para primíparas, multíparas, periodos con mayor riesgo de aborto, radiografías y ecografías positivas, etc.
